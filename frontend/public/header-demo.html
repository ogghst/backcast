Responsive Breakpoints

Desktop (>1024px): Full layout with welcome text
Tablet (768-1024px): Condensed spacing, welcome hides at 850px
Mobile (<768px): Hamburger menu with slide-in drawer

Design Features

Playfair Display (serif) for logo, DM Sans (geometric) for body. Smooth hamburger → X transform animation. Staggered reveal animations for drawer items. Body scroll lock when menu is open. Escape key to close menu.

Resize your browser window or use browser DevTools responsive mode to see the header adapt to different screen sizes.
