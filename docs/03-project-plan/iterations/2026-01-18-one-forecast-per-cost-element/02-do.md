## Temporal Query Patterns

### For `ForecastService.get_for_cost_element()`

Use the inverted FK pattern:

```python
# Query via cost_element.forecast_id (NOT forecast.cost_element_id)
stmt = (
    select(Forecast)
    .join(CostElement, CostElement.forecast_id == Forecast.forecast_id)
    .where(
        CostElement.cost_element_id == cost_element_id,
        Forecast.branch == branch,
        # Apply bitemporal filters
        Forecast.deleted_at.is_(None),  # For "current" queries
        # OR use _apply_bitemporal_filter for as_of queries
    )
)
```

### For "Current" Queries

```python
stmt = stmt.where(
    func.upper(Forecast.valid_time).is_(None),
    Forecast.deleted_at.is_(None)
)
```

### For Time-Travel Queries

```python
stmt = self._apply_bitemporal_filter(stmt, as_of)
```
