**Decisions Made:**

- **Lazy imports for extraction libs**: pypdf, python-docx, openpyxl, python-pptx are imported inside their respective functions to avoid import errors if any library has issues at module level. Also allows mypy overrides to work cleanly.
- **Callable type alias**: Used `Callable[[bytes], str]` for the extractor dict to satisfy strict mypy.
- **512 KB size limit**: Enforced in `extract_text()` with ValueError, caught in upload endpoint to return 400.
- **Null content graceful degradation**: When content is None (unsupported type or failed extraction), format_multimodal_messages produces `[User attached: filename]` placeholder.
- **Migration trimming**: Autogenerated migration included model drift from unrelated tables. Manually trimmed to only the attachment table changes.
- **Markdown support**: Added `"text/markdown": "md"` to ALLOWED_DOCUMENT_TYPES as requested.
