## Learnings

### Technical Discoveries

1. **Bitemporal FK Constraints:** Foreign key constraints cannot reference root IDs in bitemporal systems.
2. **Multi-Word Entity Names:** Custom command classes needed for `_root_field_name` overrides.
3. **TSTZRANGE Defaults:** Required for migration correctness.
4. **RBAC Configuration:** New permissions must be explicitly added to roles in `backend/config/rbac.json`.
5. **AntD Select Testing:** Playwright interactions with AntD Select require robust locators (filtering by type or specific class selectors) as virtual DOM affects option visibility.
