## Decisions Made During Implementation

1. **No helper function needed.** The `model_validate() + to_markdown()` pattern is 2 lines and appears at 3 sites. A helper would add indirection without clarity gain.
2. **Fallback handling in get_briefing tool.** Returns "No briefing available yet." for both missing and malformed `briefing_data`, matching the previous behavior.
3. **Log metric change.** Specialist completion log changed from `briefing length=%d` (character count of markdown) to `briefing sections=%d` (number of sections). More meaningful metric since markdown is no longer stored.
