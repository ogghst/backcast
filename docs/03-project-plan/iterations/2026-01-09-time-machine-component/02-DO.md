## Technical Decisions

1. **Zustand with Immer & Persist** - Following `useAuthStore` pattern
2. **React Context for Propagation** - Clean API for consuming components
3. **Query Invalidation** - Automatic refetch when time/branch changes
4. **Initial Time = Project Start** - Better UX for viewing project history
5. **Timeline = Project Range** - Not "now", actual project dates
6. **Custom Detail Hooks** - Wrapper pattern for time-travel support
