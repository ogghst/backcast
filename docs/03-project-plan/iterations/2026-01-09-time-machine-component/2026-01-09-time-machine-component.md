design a time machine component able to let user navigate through time and see how the project was evolving.
the component shall be present in the UI header when a project is selected.
it shall allow to show a timeline of the project, from start to end.
it shall allow to zoom in and out the timeline to navigate through time.
it shall allow to click on a specific point in the timeline to navigate to that point in time.
it shall show the current time as a marker in the timeline.
it shall show the branches as markers in the timeline.
it shall show the current time as marker in the timeline.
it shall be opened at the current time by default.
it shall remember the last selected time and open at that time when the project is selected again.
it shall inform the frontend of the selected time so that it can update the UI accordingly.
