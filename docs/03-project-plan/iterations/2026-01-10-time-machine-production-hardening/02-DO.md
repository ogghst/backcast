## Critical Bugs Fixed

### Bug 1: CreateVersionCommand timestamp collision ✅

**File:** `app/core/versioning/commands.py`  
**Fix:** Added explicit clock_timestamp() update after flush

```python
# Fix transaction_time to use clock_timestamp() for wall-clock time
stmt = text(
    f"""
    UPDATE {self.entity_class.__tablename__}
    SET transaction_time = tstzrange(clock_timestamp(), NULL, '[]')
    WHERE id = :version_id
    """
)
```

### Bug 2: Soft delete not time-travel aware ✅

**File:** `app/core/versioning/service.py`  
**Fix:** Check if entity deleted AFTER as_of timestamp

```python
# TEMPORAL DELETE CHECK: Entity visible if not deleted, or deleted AFTER as_of
or_(
    cast(Any, self.entity_class).deleted_at.is_(None),
    cast(Any, self.entity_class).deleted_at > as_of,
)
```

### Enhancement: Branch mode with fallback ✅

**File:** `app/core/versioning/service.py`  
**Added:** `branch` and `branch_mode` parameters to `get_as_of()`

- **STRICT mode** (default): Only return from specified branch
- **MERGE mode**: Fall back to main if not found on branch
- Respects branch deletions (no fallback if explicitly deleted)
