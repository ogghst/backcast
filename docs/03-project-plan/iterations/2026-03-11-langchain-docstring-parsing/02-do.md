**Decisions Made:**

- **LangChain tool() decorator API**: Cannot pass `name` as keyword to tool() when using decorator pattern. Override name after creation instead.
- **Type safety with LangChain**: Used `type: ignore[call-overload]` for tool() decorator due to complex overload signatures.
- **Context injection**: Used Annotated[ToolContext, InjectedToolArg] to hide context from tool schemas while injecting at runtime.
- **Backward compatibility**: Added default value for user_role in ToolContext to maintain backward compatibility.
- **RBAC integration**: RBACToolNode falls back to standard ToolNode when context is not provided.
