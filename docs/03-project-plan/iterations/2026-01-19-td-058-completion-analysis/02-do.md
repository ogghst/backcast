## Lessons Learned

1. **SQLAlchemy Async Issues**: Object expiration after updates can cause `MissingGreenlet` errors when accessing attributes. Solution: Capture IDs immediately after creation before objects expire.
2. **Relative vs Fixed Dates**: Using relative dates (e.g., `datetime.now(UTC) - timedelta(days=10)`) is more reliable than fixed dates in tests.
3. **Overlap Check Semantics**: When checking for overlaps before closing a version, use `exclude_version_id` to exclude that version from the check since it will be closed.
4. **Timestamp Generation**: Generate timestamps once in Python before using them multiple times to avoid creating empty ranges when `datetime.now(UTC)` is called twice within the same microsecond.
